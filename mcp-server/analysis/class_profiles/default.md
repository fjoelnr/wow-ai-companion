## General Analysis Focus
- **Interrupts**: Were interruptible casts handled? Check interrupt count vs. available opportunities.
- **Deaths**: What caused each death? Could defensives have prevented it?
- **Damage Uptime**: Was the player active throughout the fight, or were there idle gaps?
- **Defensive CD Usage**: Were major defensives used proactively or wasted?
- **Dispels**: Were dispellable debuffs removed promptly (if applicable)?
- **Positioning**: Did excessive damage taken suggest poor positioning?
