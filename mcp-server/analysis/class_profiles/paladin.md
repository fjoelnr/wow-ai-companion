## Paladin Analysis Focus
- **Interrupts**: Rebuke usage on interruptible casts.
- **Defensives**: Divine Shield, Ardent Defender, Guardian of Ancient Kings – proper timing.
- **Blessing of Protection/Sacrifice**: Were externals used to save others?
- **Lay on Hands**: Emergency heal – was it used when needed?
- **Avenging Wrath/Crusade**: Major CD alignment with burst windows.
- **Cleanse**: Dispel usage for removable debuffs.
- **Divine Steed**: Movement ability for mechanics.
