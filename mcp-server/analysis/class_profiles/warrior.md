## Warrior Analysis Focus
- **Interrupts**: Pummel should be used on cooldown for interruptible casts. Track interrupt rate.
- **Defensive CDs**: Shield Wall, Last Stand, Rallying Cry – were they used during dangerous moments?
- **Execute Phase**: Damage should spike below 20% HP target (Execute window). Check if Execute usage increased.
- **Rage Management**: Wasted rage = wasted damage. Check for rage-capping indicators.
- **Battle Shout**: Should be maintained as a buff for the group.
- **Spell Reflect**: Was it used against reflectable mechanics?
