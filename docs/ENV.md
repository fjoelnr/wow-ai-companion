# Environment Variables (EN)
This document describes environment variables used in the project.
See also `ENV_VARS.md` (in German).

| VARIABLE | PURPOSE | Example |
| --- | --- | --- |
| SV_PATH | Path to character SavedVariables | `/svshare/.../AICompanion.lua` |
| COMBATLOG_PATH | Path to WoW CombatLog file | `/svshare/Logs/WoWCombatLog.txt` |
| MCP_URL | URL of MCP server | `http://localhost:8080` |
| TIPS_COUNT | Number of tips to generate | `5` |
