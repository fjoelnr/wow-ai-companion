# Workflow: Migrate Repository to ANR

1. Inspect current repository structure.
2. Create global context (`AGENTS.md`).
3. Create repository map (`.agents/context-index.md`).
4. Add local context files (`*/AGENT.md`) for key directories.
5. Define reusable skills and workflows.
6. Define guardrails and restricted areas.
7. Validate ANR structure and iterate.
